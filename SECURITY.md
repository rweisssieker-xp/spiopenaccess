# Security Policy

Wenn du ein sicherheitsrelevantes Problem findest, erstelle bitte kein oeffentliches Issue mit Exploit-Details.

Stattdessen:

- beschreibe die betroffenen Dateien oder Module
- erklaere reproduzierbare Schritte
- nenne die potentielle Auswirkung

Bis ein privater Meldeweg existiert, sollten sensible Details zurueckhaltend behandelt werden.
